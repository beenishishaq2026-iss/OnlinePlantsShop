# Online Plants Shop

A professional **ASP.NET MVC** project for an online plant store with modern web features and architecture.

---

## **Project Overview**

This project simulates a full-featured online plant shop with:

- **Real-time updates** using **SignalR**  
- **Dependency Injection (DI)** for clean architecture  
- **Entity Framework & LINQ** for database management  
- **Identity with Claim & Policy-based Authorization**  
- **Custom Identity configuration** for roles & permissions  
- **Cart and Order Management** with full CRUD operations  
- **User Reviews, Comments, and Ratings** for products  

---

## **Features**

### **User Features**
- Browse plants with category filters  
- Add/remove items in cart  
- Place and track orders  
- **Submit reviews, comments, and ratings** for products  
- User authentication & authorization  

### **Admin Features**
- Add, update, or remove plants  
- Manage orders and user reviews  
- Role-based access control (Admin/User policies)  
- Real-time order and review notifications via SignalR  

### **Technical Features**
- ASP.NET MVC with clean project structure  
- DI for service management  
- Identity customization with policies and claims  
- SignalR for notifications  
- LINQ for queries and operations  
- Review system with ratings stored in the database
